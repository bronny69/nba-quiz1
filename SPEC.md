# NBA 冷知識問答網站規格說明

## 專案目標
建立一個純前端的 NBA 冷知識問答網站，可以在 GitHub Pages 上運行，讓使用者能夠測試他們對於 2010 年至今的 NBA 相關知識。

## 技術規格
- 純前端實現（HTML、CSS、JavaScript）
- 使用 CDN 版本的外部函式庫
  - Bootstrap 5.3.0（UI 框架）
- 不需要後端資料庫，所有問題都存放在前端

1. 所有外部資源都必須使用 CDN 版本，確保可以在 GitHub Pages 上正常運行
